## 1. 문제 정의

취업을 준비하면서 어떤 스펙을 준비해야 하는지 잘 모르겠고, 내가 원하는 분야의 채용공고와 공모전 일정을 직접 확인하는게 귀찮다. CareerFit AI로 원하는 직무의 필요 스펙과 채용공고, 공모전 일정을 알려준다. 그리고 나의 상태를 분석해서 필요한 공보전 이력이나 보완할 스펙을 알려준다.



## 2. 사용자 정의

- 대상: 취업 준비중인 대학교 3~4학년, 공모전이 처음인 학생



- 사용 시나리오:

1. 사용자가 전공, 보유 스킬, 관심 직무를 입력한다.

2. CareerFit AI가 데이터를 검색해 역량 매칭 결과를 반환한다.

3. 어떤 데이터를 기반으로 답변했는지 출처를 함께 표시한다.



## 3. 데이터 계획

- 취업 공고 컬럼: 회사명, 직무명, 우대사항, 지원자격, 업무 내용, 회사 위치, 채용공고 일정

- 공모전 컬럼: 공모전 제목, 지원 자격, 공보전 내용, 공모전 기간

- 데이터 출처: 강사 제공 목업 CSV (3일차)
